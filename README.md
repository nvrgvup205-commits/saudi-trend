# سعودي تريند - صفحة الهبوط

صفحة هبوط احترافية لشركة **سعودي تريند** المتخصصة في التسويق وتقديم الحلول والخدمات للشركات والمؤسسات.

## المميزات

- تصميم عصري ومتجاوب مع جميع الأجهزة
- دعم كامل للغة العربية واتجاه RTL
- أقسام: الرئيسية، من نحن، الخدمات، لماذا نحن، التواصل
- تأثيرات حركية وتفاعلية
- نموذج تواصل تفاعلي

## التشغيل

افتح ملف `index.html` مباشرة في المتصفح، أو استخدم خادم محلي:

```bash
# باستخدام Python
python3 -m http.server 8000

# أو باستخدام npx
npx serve .
```

ثم افتح [http://localhost:8000](http://localhost:8000) في المتصفح.

## هيكل المشروع

```
├── index.html      # الصفحة الرئيسية
├── css/
│   └── style.css   # التنسيقات
├── js/
│   └── main.js     # التفاعلات
└── README.md
```

## التخصيص

- **الألوان**: عدّل متغيرات CSS في `:root` داخل `css/style.css`
- **المحتوى**: عدّل النصوص مباشرة في `index.html`
- **بيانات التواصل**: حدّث الهاتف والبريد في قسم التواصل
